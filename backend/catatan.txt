mengintas tool TS (Type Script untuk Backend)
npm install -D typescript ts-node @types/node @types/express @types/cors
Huruf -D artinya kita menginstal ini hanya untuk alat bantu saat coding (Development).@types/... 
adalah paket khusus yang memberikan VS Code "buku panduan" tentang isi perintah Express dan Cors, agar fitur anti-typo bisa bekerja dengan maksimal.
